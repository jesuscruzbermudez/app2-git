Información sobre el Perfil de Egreso de la Especialidad


La información que ofrece la carrera de Técnico en Programación permite al egresado a través de la articulación de saberes de diversos campos, realizar actividades dirigidas a la:

  • Instalación y desarollo de software de aplicación utilizando programación estructurada y orientada a objetos en ambientes web y móviles, con almacenamiento persistente de datos,
  • Así como la configuración y administración de plataforma e-learing y
  • Comercio electrónico
Durante el proceso de formación de los cinco módulos, el estudiante desarollará o reforzará:
Las siguentes competencias profesionales:
  • Desarolla e instala software de apliación utilizando programación estructurada, con almacenamiento persistene en los datos
  • Desarolla software de apliación utilizando programación orientada a objetos, con almacenamiento persistene de los datos
  • Desarolla aplicaciones web y móviles
  • Administra sistemas operativos, aplicaciones y servicios
  • Desarolla, administra y configura soluciones de e-learning y comercio electrónico

  • Modulo I

    Desarolla e instala software de aplicación utilizando programación estructurada, con almacenamiento persistente de los datos

  • Modulo II

    Desarolla software de apliación utilizando programación orientada a objetos, con almacenamiento persistene de los datos

  • Modulo III

    Desarolla aplicaciones web y móviless

  • Modulo IV

    Administra sistemas operativos, aplicaciones y servicios

  • Modulo V

    Desarolla, administra y configura soluciones de e-learning y comercio electrónico
